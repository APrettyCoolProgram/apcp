***

All `.cs` files should have a file header comment.

```csharp
// =============================================================================
// https://github.com/GithubAccount/project-name
// uYYMMDD_code
// uYYMMDD_documentation
// =============================================================================
```

For example:

```csharp
// =============================================================================
// https://github.com/GithubAccount/project-name
// u251120_code
// u251120_documentation
// =============================================================================
```
